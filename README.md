## What I build

- Crypto and prediction-market trading bots
- Polymarket research, execution, and monitoring systems
- AI-agent workflows for trading, research, and operations
- Data dashboards, automation scripts, and decision-support tools
- Python-first analytics with practical risk controls

## Current focus

- Building safer paper-mode and live-mode trading infrastructure
- Improving signal quality with EV filters, Kelly sizing, and simulations
- Turning manual crypto research into repeatable agent workflows
- Shipping small, useful tools that help produce income, not noise
